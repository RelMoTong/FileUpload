# 版本摘要（精简版）

## v3.2.0 (2025-12-17)
- **文件清理功能重大升级**：
  - 结果区从日志框升级为可勾选表格（文件名/路径/大小/修改时间），支持排序、搜索
  - 右键菜单：打开所在文件夹、复制路径
  - 扫描可取消 + 实时进度（当前目录/文件数/累计大小）
  - 删除前显示清单摘要 + Top 5 最大文件
  - 安全默认：回收站默认勾选，提前提示可用性
  - 新增过滤条件：按修改天数筛选文件
  - 复选框联动：监控/自定义文件夹勾选时才启用输入框
  - 布局优化：使用 QSplitter，可调整大小，最小 1000x700
  - 文案改进：明确定位为"按目录和扩展名清理"，非系统级磁盘清理
- 详细说明见 `docs/CLEANUP_UPGRADE.md`

## v3.1.1
- 磁盘清理：扫描/删除改为异步线程，支持回收站（send2trash 可选），国际化补齐，滚动区域避免窗口铺满。
- 版本号单一来源：`src.__version__`，PyInstaller spec 读取生成产物名。

## v3.1.0
- SMB/FTP 模式重构，新增 FTP 服务器开关、密码可见性切换。
- 启动时依赖检查；UI/worker 模块化；保留兼容入口。

## v3.0.x
- 项目模块化，精简旧入口为 `pyqt_app.py` 包装。
- UI 细节、日志与权限优化。

## v2.x（概览）
- 引入协议选项、权限与自动备份；磁盘清理早期说明与性能报告。

说明
- 当前架构与使用请见 `docs/README.md`；历史零散文档已合并/删除。
