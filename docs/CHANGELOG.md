# 版本摘要（精简版）

## v3.2.0 (2026-04-13)
- 文件清理界面升级为可勾选表格，支持排序、搜索、右键操作、扫描进度、删除前摘要和更清晰的交互反馈。
- 自动清理配置补齐到界面层，支持保留天数、格式过滤、回收站删除模式，并与主窗口配置保存/回填链路统一。
- 自动清理执行链路调整为由主窗口统一调度，Worker 仅负责告警；紧急清理模式会透传到统一清理任务。
- 权限与按钮联动收敛：`guest` 不能执行清理和配置保存，运行中禁止修改配置，删除按钮只跟随勾选文件启用。
- 配置加载守卫补齐，避免启动时把配置回填误记为“用户手动修改”。
- 打包链路对齐版本号，PyInstaller 固定使用 `PySide6`，并补充 `Send2Trash` 依赖，保证回收站功能在打包版中可用。
- 详细说明见 `RELEASE_NOTES_v3.2.0.md`、`docs/GITHUB_RELEASE_v3.2.0.md` 和 `docs/CLEANUP_UPGRADE.md`。

## v3.1.1
- 磁盘清理：扫描/删除改为异步线程，支持回收站（send2trash 可选），国际化补齐，滚动区域避免窗口铺满。
- 版本号单一来源：`src.__version__`，PyInstaller spec 读取生成产物名。

## v3.1.0
- SMB/FTP 模式重构，新增 FTP 服务器开关、密码可见性切换。
- 启动时依赖检查；UI/worker 模块化。

## v3.0.x
- 项目模块化。
- UI 细节、日志与权限优化。

## v2.x（概览）
- 引入协议选项、权限与自动备份；磁盘清理早期说明与性能报告。
