╚════════════════════════════════════════════════════════════╝

【启动方式】
  Windows:  双击 启动程序.bat
  命令行:  python main.py

【核心功能】
  ✓ 异步上传 - 避免网络阻塞
  ✓ 自动备份 - 上传后移动源文件
  ✓ 日志记录 - 详细记录所有操作
  ✓ 连接检测 - 自动检测网络状态
  ✓ 配置保存 - 自动保存用户设置

【操作流程】
  1. 点击"浏览"选择源文件夹
  2. 点击"浏览"选择目标文件夹（网络共享）
  3. 点击"浏览"选择备份文件夹
  4. 选择监控模式（定时/实时）
  5. 设置上传间隔（10-3600秒）
  6. 点击"测试连接"验证网络
  7. 点击"保存配置"保存设置
  8. 点击"开始上传"启动服务

【按钮说明】
  开始上传 - 启动上传服务
  停止上传 - 停止上传服务
  测试连接 - 测试目标文件夹连通性
  保存配置 - 保存当前配置到文件
  清空日志 - 清空界面日志显示

【监控模式】
  定时上传 - 按间隔时间定期检查（推荐）
  实时监控 - 持续监控文件夹变化

【推荐设置】
  网络良好: 间隔30秒，定时上传
  网络一般: 间隔60秒，定时上传
  网络繁忙: 间隔120秒，定时上传

【支持格式】
  JPG, JPEG, PNG, BMP, TIFF, GIF

【文件位置】
  日志文件: logs/upload_YYYYMMDD.log
  配置文件: config.json

【网络路径示例】
  \\192.168.1.100\共享文件夹\图片
  \\SERVER01\Production\Photos
  Z:\Network\Upload

【常见问题】
  Q: 提示"目标文件夹不存在"？
  A: 检查网络连接和路径是否正确

  Q: 上传速度慢？
  A: 增加上传间隔时间

  Q: 软件无响应？
  A: 停止后重新启动

【技术规格】
  开发语言: Python 3.6+
  GUI框架: tkinter（Python标准库）
  运行环境: Windows/Linux/macOS
  网络支持: SMB/CIFS共享协议

【安全特性】
  ✓ 文件完整性校验
  ✓ 错误自动处理
  ✓ 重复文件检测
  ✓ 详细日志记录
