## 🚀 使用流程

### Step 1：选择扫描范围（左侧）
1. 切换到"基础设置" Tab
2. 勾选目标文件夹
3. 点击文件夹旁的按钮操作路径
4. 选择文件格式（或使用预设）

### Step 2：设置过滤条件（可选）
1. 切换到"高级设置" Tab
2. 勾选"仅显示/删除 N 天前"
3. 输入自定义扩展名（逗号分隔）

### Step 3：扫描文件
1. 点击左侧底部"🔍 开始扫描"
2. 查看顶部摘要条确认条件
3. 观察右侧进度条和实时信息
4. 需要时点击"⏹ 取消"中止

### Step 4：筛选和确认
1. 使用搜索框查找特定文件
2. 点击 Chip 按钮快速筛选
3. 取消勾选不需要删除的文件
4. 查看底部统计信息

### Step 5：执行删除
1. 点击删除按钮旁的下拉箭头
2. 选择"移入回收站"或"永久删除"
3. 点击"🗑️ 删除选中"
4. 确认清单摘要和 Top 5 文件
5. 等待删除完成

## ⚡ 快捷操作

### 表单行按钮
- 📁 **浏览**：选择文件夹
- 🔗 **打开**：在文件管理器中打开
- 📋 **复制**：复制完整路径

### 快捷筛选
- **✓ 仅已选**：只显示勾选的文件
- **📦 大文件**：只显示 >10MB 的文件
- **🕐 最近7天**：只显示最近7天修改的文件

### 格式预设
- **全选**：选中所有格式
- **清空**：取消所有格式
- **图片**：只选图片格式

## 💡 使用技巧

### 技巧 1：组合筛选
同时启用多个 Chip 可以精确定位：
```
✓ 仅已选 + 📦 大文件
= 查看已勾选的大文件
```

### 技巧 2：路径确认
点击文件夹行的 🔗 按钮，在文件管理器中确认路径是否正确。

### 技巧 3：批量取消
使用快捷筛选 + "取消全选"可以批量操作：
1. 点击"📦 大文件"筛选
2. 点击"取消全选"
3. 取消筛选，只删除小文件

### 技巧 4：查看摘要
扫描前后对比顶部蓝色摘要条，确认条件是否符合预期。

### 技巧 5：安全删除
始终使用"移入回收站"模式，可在系统回收站恢复误删文件。
