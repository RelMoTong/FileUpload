# v3.2.0

发布日期：2026-04-13

## What's Changed

- 文件清理工具升级为表格化结果视图，支持勾选、搜索、排序、右键菜单和删除前摘要。
- 自动清理配置补齐保留天数、格式过滤和回收站模式，配置保存与回填链路完整打通。
- 自动清理统一由主窗口调度，Worker 只负责空间不足告警，紧急模式可透传到清理任务。
- 权限与按钮联动修正，`guest` 与运行中状态下的交互行为更一致。
- 配置加载增加守卫，避免启动时误报“配置已修改”。
- 打包链路固定使用 `PySide6`，并补充 `Send2Trash`，发布版功能与源码版保持一致。

## Highlights

- 更安全的文件删除交互
- 更完整的自动清理配置能力
- 更一致的权限与按钮行为
- 更稳定的 Windows 打包产物

## Packaging

- 版本号来源：`src.__version__`
- 当前版本：`3.2.0`
- 打包输出目录：`dist/ImageUploadTool_v3.2.0/`

## Notes

- 建议发布前保留一轮打包版启动自测。
- 自动清理执行链路已有测试覆盖，但执行器接口兼容性还有一个待继续收口的回归点。
